List Organisation

Here is an example with a list of options. If you wish to move the first option to the end of the list, this is a one possible solution.


lst = [
  "Option1", 
  "Option2", 
  "Option3", 
  "Option4", 
  "Option5", 
  ]
indx = lst.index("Option1")
item = lst.pop(indx)
lst.append(item)

print(lst)

--------------------------------------------
$ ['Option2', 'Option3', 'Option4', 'Option5', 'Option1']
  1. A list named lst has been defined with values "Option1", "Option2", "Option3", "Option4", and "Option5".
  2. To identify the index of "Option1" within the list, we use the index() method and assign the result to a variable called indx. Since "Option1" is the first item in the list, the value of indx will be 0.
  3. Using the pop() method, we remove the item at the index indx (which is 0) from the list and assign it to a new variable called item
  4. Finally, we add the item (which was "Option1") back to the end of lst using the append() method.
  5. The updated lst can be printed.

    6. This method will work with any index value.
